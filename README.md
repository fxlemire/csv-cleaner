# CSV CLEANER

This tool allows to clean CSV files by removing columns and entries that solely contain null values such as `null`, `''` or `0`.

## How To Use

```bash
npm install
node clean-csv.js <PATH_TO_CSV_FOLDER>
```

Run `node clean-csv.js --help` for more info.
